# Sistem Pelaporan Rumah Sakit (SIRS 6.3)

Aplikasi pelaporan internal rumah sakit berbasis Next.js sesuai dengan standar JUKNIS SIRS Revisi 6.3 - Kementerian Kesehatan RI.

## Cara Menjalankan Aplikasi

1.  **Masuk ke direktori aplikasi**:
    ```bash
    cd sirs-app
    ```

2.  **Instal dependensi**:
    ```bash
    npm install
    ```

3.  **Jalankan server pengembangan**:
    ```bash
    npm run dev
    ```

4.  **Buka di browser**:
    Akses [http://localhost:3100](http://localhost:3100)
